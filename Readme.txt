Dataset
Human Activity Recognition Using Smartphones, from http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones

The purpose:
=========================================
Is to extract the data from the weblink, and prepare it for analysis.

The dataset includes the following files:
=========================================
- Readme.txt
- courseProject.R: script and simple explanation of the steps.
- Codebook.txt: shows details of the code.
